# Security Policy

## Reporting Vulnerabilities

If you discover a security vulnerability in this project, please report it responsibly via email:

**security@kodeme.io**

Do **not** open a public GitHub issue for security vulnerabilities.

Please include as much detail as possible:

- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (if any)

## Response Timeline

- **Acknowledgment**: Within 48 hours of receiving your report.
- **Fix timeline**: Based on severity:
  - **Critical**: Patch released within 7 days.
  - **High**: Patch released within 14 days.
  - **Medium/Low**: Addressed in the next scheduled release.

## Main Security Policy

For the overarching security policy covering all Kodeme.io projects, please refer to the [SECURITY.md in kodemeio-core](https://github.com/kodemeio/kodemeio-core/blob/main/SECURITY.md).
